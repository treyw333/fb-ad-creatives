# Product

A browser-based synchronized presentation app for a talk on optimizing Facebook ads with AI tools. One presenter controls slide navigation; up to 50 viewers follow along in real-time via Supabase Realtime.

The presentation is a continuous horizontal canvas — 14 slides laid out side-by-side in a single flex strip. Navigation pans the viewport across the strip, creating a seamless panoramic experience rather than discrete slide transitions.

Key interactive features:
- Dynamic prompt builder (Slide 9) where viewers type their own product/audience/industry values and see prompts update live before copying
- Copy-to-clipboard buttons for AI prompts
- External links to AI tools (ChatGPT, Claude, Gemini, Recraft.ai, ZenduxAI)
- Example ad creative mockups

Visual identity: premium dark futuristic theme (#0a0a0f–#12121a), restrained accents (electric blue #00d4ff, violet #8b5cf6), ambient gradient orbs, no cliché AI imagery.

Target: digital agency owners learning to use AI for Facebook ad creative production.
